# AutomationText 

AutomationText.py is a Python-based automation script that allows users to send WhatsApp messages automatically using the PyWhatKit library.

## About the Project

This project demonstrates how to automate WhatsApp message sending using Python. It can be used for:

- Sending scheduled WhatsApp messages
- Automating repetitive messaging tasks
- Learning Python automation concepts
- Understanding how third-party libraries like PyWhatKit work

## Technologies Used

- Python 3
- PyWhatKit

## Project Structure

AutomationText.py -> Main automation script  
.gitignore -> Ignored files configuration  

## How to Run

1. Install required library:
   pip install pywhatkit

2. Run the script:
   python AutomationText.py

## Note

- Make sure you are logged into WhatsApp Web.
